请确保你现在已经打开了命令行（终端），并且通过 cd 命令进入了你的博客文件夹（比如 cd myblog）。

发生了什么？ Hexo 在你的博客目录下的 source/_posts 文件夹里，自动生成了一个名为 如何发布第一篇博客.md 的文件。

.md 后缀代表 Markdown 格式，这是写博客的标准格式。

第二步：编辑内容（写作的核心）

现在，不要在终端里操作了。去你的电脑文件夹里，找到 source/_posts/如何发布第一篇博客.md 这个文件。

用什么打开它？

推荐： VS Code（程序员首选）、Typora（所见即所得，体验极佳）、或者记事本（不推荐，但能用）。

⚠️ 警告： 哪怕你把正文全删了，也千万不要删掉这两行 — 和里面的内容！这是 Hexo 识别文章标题和日期的身份证。

开始写正文： 在第二个 — 下面，你可以开始写正文了。Hexo 使用 Markdown 语法。

小白速成 Markdown 语法：

标题： 输入 # 一级标题，## 二级标题

加粗： 输入 我想加粗

列表： 输入 - 第一点

链接： 显示文字

图片：

  1. 如何删除一篇文章？
    非常简单粗暴：直接在电脑文件夹 source/_posts 里把那个 .md 文件删了。然后执行 hexo clean -> hexo g -> hexo d，网上那篇文章就消失了。

把你电脑里的图片上传到“路过图床”或“SM.MS”等免费图床网站。
